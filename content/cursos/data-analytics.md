# Data Analytics

## Información General

- **Institución**: Upgrade Hub, España
- **Fecha de finalización**: 19 de febrero de 2024
- **Duración**: 350 horas lectivas
- **Certificación**: Sí

## Descripción del Curso

Este bootcamp intensivo de Data Analytics proporciona una formación completa en el análisis y visualización de datos, con un enfoque práctico orientado a la industria.

## Contenido del Curso

### Módulo 1: Fundamentos de Análisis de Datos
- Introducción al análisis de datos
- Estadística descriptiva e inferencial
- Metodologías de análisis

### Módulo 2: Herramientas de Análisis
- Excel avanzado para análisis de datos
- SQL para consulta y manipulación de datos
- Python para análisis de datos (Pandas, NumPy)

### Módulo 3: Visualización de Datos
- Principios de visualización efectiva
- Herramientas: Tableau, Power BI
- Creación de dashboards interactivos

### Módulo 4: Business Intelligence
- KPIs y métricas de negocio
- Análisis de tendencias y patrones
- Toma de decisiones basada en datos

### Módulo 5: Proyecto Final
- Desarrollo de un proyecto completo de análisis
- Presentación de resultados y recomendaciones
- Implementación de soluciones basadas en datos

## Habilidades Adquiridas

- Análisis exploratorio de datos
- Limpieza y preparación de datos
- Visualización avanzada
- Interpretación de resultados estadísticos
- Comunicación efectiva de hallazgos
- Implementación de soluciones de Business Intelligence

## Aplicación Práctica

Los conocimientos adquiridos en este bootcamp me han permitido desarrollar análisis más profundos y efectivos en mi trabajo, especialmente en la identificación de patrones y tendencias que apoyan la toma de decisiones estratégicas.
